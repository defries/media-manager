== Description ==

Automatically removes all image attachments from posts. This is still in the alpha stage.


== Installation ==

After you've downloaded and extracted the files:

1. Upload the complete 'media-manager' folder to the '/wp-content/plugins/' directory OR install via the plugin installer
2. Activate the plugin through the 'Plugins' menu in WordPress
